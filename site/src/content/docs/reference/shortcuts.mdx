---
title: Keyboard shortcuts
lead: "Nexus is built to be driven from the keyboard. Global navigation and the review queue both have first-class shortcuts."
group: Reference
order: 2
---

## Global

<Keys>
  <KeyRow desc="Global search — jump to any project or run"><kbd>Ctrl</kbd> <kbd>K</kbd></KeyRow>
  <KeyRow desc="Start a new run">New run button</KeyRow>
  <KeyRow desc="Switch between Workbench and Review">title-bar tabs</KeyRow>
</Keys>

## Review queue

<Keys>
  <KeyRow desc="Navigate to next / previous file"><kbd>J</kbd> <kbd>K</kbd></KeyRow>
  <KeyRow desc="Approve and jump to next"><kbd>A</kbd></KeyRow>
  <KeyRow desc="Request changes"><kbd>R</kbd></KeyRow>
  <KeyRow desc="Skip this file"><kbd>S</kbd></KeyRow>
  <KeyRow desc="Jump to next unreviewed"><kbd>N</kbd></KeyRow>
</Keys>

## Sessions

<Keys>
  <KeyRow desc="Copy session details">right-click</KeyRow>
</Keys>

<Callout type="note">
On macOS, use <kbd>⌘</kbd> wherever <kbd>Ctrl</kbd> is shown — for example <kbd>⌘</kbd>+<kbd>K</kbd> for global search.
</Callout>
