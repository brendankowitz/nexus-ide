## Anatomy

The rail lists every registered project, grouped and ordered however you like. Each entry shows the project name, current branch, and a set of status badges.

## Reading the badges

| Badge | Meaning |
|-------|---------|
| `4` | Uncommitted changes — dirty file count. |
| `1wt` | Worktrees attached to the project. |
| `1ag` | Agents currently running against the project. |

A glance down the rail tells you which repos are dirty, which have parallel work in flight, and where agents are active.

## One-click switching

Click any entry to rebind the whole app to that project. Branches, worktrees, sessions, and review all follow. No window juggling, no re-opening folders.

## Managing density

With a large fleet, the rail can get long. You can:

- **Collapse groups** to hide projects you're not touching today.
- **Drag** projects to reorder them or move them between groups.

<Callout type="tip">
Press <kbd>Ctrl</kbd>+<kbd>K</kbd> to jump straight to any project by name — faster than scrolling the rail once you have a dozen repos registered.
</Callout>
